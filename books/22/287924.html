    &nbsp;    &nbsp;    &nbsp;    &nbsp;  “女弟子？我一共有十二大女弟子，这件事人所共知！”弥弘刚双手背在身后。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  “既然弘刚长老有女xing弟子，请问你这些弟子是如何保持清白的？是不是你只要单独把她们叫过去传授技艺，我就可以认定你们不清白？”易青淡淡说道。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  “哈哈，弥弘刚，之前我一直想不通你为何要招收这么多女弟子，现在易青圣女一句话就还原了真相，哎，身为一个宗门太上长老，竟然做出如此卑鄙的事，真不知道该怎么说你！”  &nbsp;    &nbsp;    &nbsp;    &nbsp;  “当你的女弟子就没了清白，这难道就是你的潜规则？弥弘刚你真是丢尽了八大宗门的脸面……”  &nbsp;    &nbsp;    &nbsp;    &nbsp;  “丢人啊！我以前一直不理解什么叫做人面兽心，现在终于知道了，这就叫人面兽心，而且还是极其厉害的手心！禽兽不如啊！”  &nbsp;    &nbsp;    &nbsp;    &nbsp;  许游、张文顺、韩同新再次放声大笑。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  他们三人刚才被弥弘刚侮辱，实力不济，没办法报仇，现在听易青字字珠心，一句话就让对方搬起石头砸自己的脚，全部笑了起来，说不出的畅快。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  “你……”听到这话，弥弘刚脸sè一阵红一阵白，差点没活活气死，不过他就算在嚣张也不敢对一个女弟子动手，真这样的话，不但有损身份，还会被人误认为被易青说中了事实，恼羞成怒，名声更臭！  &nbsp;    &nbsp;    &nbsp;    &nbsp;  咬了咬牙，接着道：“少在这胡言乱语，我收女徒弟只是为了传艺，怎么可能会有那种想法？你这样故意混绕视听是不想说你和聂云的关系，还是不想承认你和冰清玉洁已经没有任何关系了？”  &nbsp;    &nbsp;    &nbsp;    &nbsp;  “弘刚长老没直接回答我的问题，那我是不是可以认为，你王顾左右而言他，是故意岔开话题，不想讨论你和你女徒弟不可告人的秘密？如果真是清白，你又为何不直接说，非要装出受了委屈的样子，愤怒不已？”  &nbsp;    &nbsp;    &nbsp;    &nbsp;  易青毫不在乎，随口说道。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  一路跟聂云一起，多次逃离弥神宗的围杀，在她心中高高在上的八大宗门也不过如此，心中再无畏惧，所以说起话来也没什么可怕的。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  “我堂堂弥神宗纳虚境巅峰的太上长老竟然被你一个小丫头难住，好，好！果然伶牙俐齿，既然如此，我现在弥弘刚就向天道发誓，如果和我的女弟子有任何不清楚的关系，我会遭到天道制裁，永世不得超生！好了，你如果承认和那个聂云没任何关系，也没有任何感情，就发下天道誓言，这样我就相信，你们二人清白，你是冰清玉洁！”  &nbsp;    &nbsp;    &nbsp;    &nbsp;  听到女孩直刺要害的话语，弥弘刚脸sè变了变，一声长啸竟然发下了天道誓言。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  天道誓言是一种最严酷的誓言，发誓者，会遭到天道的监督，一旦违背，天道就会将其灭杀，毫无回旋余地！  &nbsp;    &nbsp;    &nbsp;    &nbsp;  按照道理弥弘刚这样的强者就算被杀也绝不会发现这种誓言，可现在他被易青将军，弄得实在没办法，只好用这种誓言证实清白，不然，真传出去，自己一世英名将会尽随流水！  &nbsp;    &nbsp;    &nbsp;    &nbsp;  本来想弄得易青非常难看，让她愧疚，没有资格成为玲珑圣女，现在却没想到被她摆了一道，发下重誓，这种事对堂堂纳虚境巅峰强者的侮辱实在太大了。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  但现在人在虎背，没有后路可走，满身是嘴也说不清楚，只能这样做了。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  “好啊，发个天道誓言而已，没什么！”见对方连天道誓言都能发出来，易青也知道将其逼得了绝路，淡淡一笑“我现在对天道发誓，我和聂云之间到目前为止，没有任何超出朋友的不正当关系，天地可鉴，如有假话，天诛地灭！”  &nbsp;    &nbsp;    &nbsp;    &nbsp;  发誓的时候，易青留了个心眼，故意说“到目前为止”，强调了时间限制，至于以后的事，以后再说！  &nbsp;    &nbsp;    &nbsp;    &nbsp;  虽然这一路，她和聂云同在一个房间，有些不便，但到现在，自己的确和他没发生任何不正当的关系，这点就算发下天道誓言也没什么。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  唯一的暧昧也就是自己洗澡被对方看了个jing光，这点在天道誓言下，无伤大雅。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  “怎么样，弥弘长老，我该发的誓也发了，你该问的也问了，我现在能继续接受册封吗？”发完誓言，易青淡淡一笑，问道。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  “这……好，你继续接受册封吧，我再无意见！”弥弘长老气呼呼的坐了下来。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  他怎么都没想到自己本来想要弄得易青无法成为圣女，结果自己却被弄得颜面扫地。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  “这个弥弘刚没病吧，一男一女共处一室就非要发生关系吗？自己思想龌龊，也以为别人和他一样！”  &nbsp;    &nbsp;    &nbsp;    &nbsp;  “真是无耻，堂堂弥神宗太上长老，这样不要脸也太掉价了吧！”  &nbsp;    &nbsp;    &nbsp;    &nbsp;  “你没听刚才他发的那个天道誓言，根本就含糊不清，什么‘和我的女弟子有任何不清楚的关系’，这个不清楚的关系指的什么？如果他把猥亵女弟子也当成清楚关系，人人都知道，这天道誓言岂不一点作用都没有？”  &nbsp;    &nbsp;    &nbsp;    &nbsp;  “我看他和女弟子的关系还是怪怪的，不说其他，就他这种猥琐思想，一男一女在一起就想着不纯洁，能有纯洁关系才怪！”  &nbsp;    &nbsp;    &nbsp;    &nbsp;  弥弘长老刚说完不在追究，就听到下面玲珑仙宗的诸多弟子悄悄嘀咕起来。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  她们嘀咕的声音都不小，每一句话都清晰传到弥弘长老的耳朵中，让他脸sè变得难看，差点没一口鲜血喷出活活气死。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  “玲珑仙宗的弟子果然有见识，嗯，今天这件事我们看来必须向宗门禀报，不然堂堂八大宗门，名门正派，猥亵女徒弟，弄潜规则……这种事传出去，实在有损八大宗门名声！”  &nbsp;    &nbsp;    &nbsp;    &nbsp;  “是啊，这种是必须严惩，不然，咱们八大宗门和藏污纳垢之处还有和区别？”  &nbsp;    &nbsp;    &nbsp;    &nbsp;  “我觉得有些人思想本来就不纯正，光想着孤男寡女，都一把年纪了，还整天胡思乱想，真不知道这一千多岁怎么活过来的！八大宗门有这样一朵奇葩，还真够丢人的……”  &nbsp;    &nbsp;    &nbsp;    &nbsp;  许游、张文顺、韩同新也满脸带着笑意，自言自语似地说道。
