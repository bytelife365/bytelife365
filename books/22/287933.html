第五百零五章 玲珑宗主的请求 上

    &nbsp;    &nbsp;    &nbsp;    &nbsp;  武技到了高深处，不光是简单的招数，最重要的是意境和对大道的理解，这招风萧秋杀就是这种情况，自己现在没有这种特殊的剑招意境，强行修炼，没有几年功夫，别说大成，小成都没有可能！  &nbsp;    &nbsp;    &nbsp;    &nbsp;  除非自己是剑道师！  &nbsp;    &nbsp;    &nbsp;    &nbsp;  剑道师，特殊天赋之一，在天眼师前面，排行第十七，拥有这种天赋，对剑术有特殊的理解，能上瞬间领悟各种剑招的意境，哪怕再厉害的剑术，只要看上一眼，就能立刻融会贯通，施展出来！  &nbsp;    &nbsp;    &nbsp;    &nbsp;  自己没有这种天赋，虽然拥有两世为人经历，对剑术理解远超一般人，可想要很快练成风萧秋杀这种强大的剑法，还是力有不逮。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  传说中，剑神宗的创派祖师就是位剑道师，领悟自己的剑道，如果自己有机会混入剑神宗，感受剑道之气，成为剑道师，在修炼这些剑法就简单了。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  “琉璃之魂，纯净如水，炼化尸体！”  &nbsp;    &nbsp;    &nbsp;    &nbsp;  将脑中乱七八糟的想法甩开，聂云灵级后期的灵魂疯狂向弥弘刚的尸体上灌输进去，开始炼化尸体。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  尸体级别越高，越难炼化，当初聂云炼化十二头天桥境尸体就耗费了足足一天一夜，弥弘刚更强，同样时间，恐怕根本完成不了。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  不过，聂云也不愁，反正有的是时间，纯净透明的琉璃之魂在弥弘刚的尸体内不停灼烧，进行炼化。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  刚才他将这个尸体检查了一遍，弥弘刚虽然是弥神宗的太上长老，却没太多宝物，最珍贵的就是那枚破空境符箓，剩下珍贵的就是些绝品灵石，至于其他东西，一穷二白，真对不起他这里厉害的修为和身份。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  按照自己的想法，他这种人，身上各种丹药、灵兵、药材应该数不胜数，搜刮一番也好暴富，怎么都想不到，除了符箓和灵石，比自己都穷。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  真不知道弥神宗到底怎么回事，一个个高层，都不富裕，反正自己杀了这么多长老，宝贝没得到多少。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  滋滋滋滋！  &nbsp;    &nbsp;    &nbsp;    &nbsp;  炼化尸体非常枯燥也很麻烦，不知过了多长时间，聂云缓缓睁开了眼睛。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  “终于炼化了，不过，我现在的灵魂只有灵级后期，没达到巅峰，根本驱动不了这个尸体……”  &nbsp;    &nbsp;    &nbsp;    &nbsp;  灵魂融入尸体，聂云感受了一下，随即哭丧着脸，没了刚才的兴奋。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  弥弘刚的尸体是厉害，可自己的灵魂只有灵级后期，根本没办法驱动发挥出尸体的全部力量。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  想要晋级纳虚境，灵魂首先要达到灵级巅峰，聂云只有灵级后期灵魂，虽然同级别中算是强的，可想要完美控制弥弘刚的尸体，就不行了。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  “我现在的灵魂力量，最多能将其控制的行走，战斗起来恐怕连那个弥星的尸体都不如……”  &nbsp;    &nbsp;    &nbsp;    &nbsp;  知道这个尸体的战斗力，聂云摇了摇头，一阵无奈。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  灵魂就好像发动机，汽车发动机在汽车上面力量很大，放在飞机里就不行了！发动机不行，就算飞机再厉害，也飞不起来啊！  &nbsp;    &nbsp;    &nbsp;    &nbsp;  现在聂云的情况就是这样，尽管炼化了弥弘刚的尸体，可自身的灵魂力量不够，根本不能发挥出纳虚境巅峰的战斗力，甚至真正战斗起来，连天桥境巅峰的弥星尸体都不如！  &nbsp;    &nbsp;    &nbsp;    &nbsp;  不能飞的飞机，连步行都不如，更别说汽车了！  &nbsp;    &nbsp;    &nbsp;    &nbsp;  “必须想办法炼制丹药晋升灵魂修为了，灵魂强了实力才强……”  &nbsp;    &nbsp;    &nbsp;    &nbsp;  现在快速晋级灵魂的方法就是炼制出绝世丹，可惜炼制这种丹药还差一样金株草，去哪里寻找都不知道。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  “嗯，炼化个尸体竟然过了三天三夜，玲珑宗主都找了我好几遍了……”  &nbsp;    &nbsp;    &nbsp;    &nbsp;  想了一下，知道没有金株草难以练成绝世丹，当下也就不多想，聂云站起身来。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  炼化弥弘刚的尸体时间过了整整三天三夜，这期间玲珑宗主过来几次，当看到自己布置下的禁制，知道自己可能在修炼，也识趣的没继续观察。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  虽然这次自己进入紫华洞府，剑法没练成，炼化了尸体又不能用，实力没增长多少，体内暴增的实力却巩固了不少，让自己气息更加沉稳，灵魂更加凝聚。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  呼！  &nbsp;    &nbsp;    &nbsp;    &nbsp;  身体一晃，就从紫华洞府走了出来，大手一抓就将自己布置下的禁制撤掉。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  刚撤销禁制，就听到空中呼呼风声，玲珑宗主再次飞了过来。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  这次霍颖和她一起，二人同时落在院子中，白衣胜雪，宛如两个仙子降临，美艳不可方物。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  “你出关了，之前我来了好几次，看到你布下禁制，知道肯定在修炼，就没打扰！”看到聂云，玲珑宗主笑着说道。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  聂云本身实力不值得她这样，但少年的潜力以及他师父，就算她再小心一些，也不算过分。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  “见过玲珑宗主，不知宗主找聂云有什么事？”聂云向前迎了上去。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  “据我所知化云宗一共有东、南、西、北四大丹田穴窍境的无上长老，不知你师父是哪位前辈，可否告知……”  &nbsp;    &nbsp;    &nbsp;    &nbsp;  玲珑宗主犹豫了一下，问道。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  “呵呵，我师父他老人家不愿意让我把他的真正身份告诉别人，所以……还请宗主见谅！”聂云淡淡一笑。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  自己这个师父是假的，虽然这四大高手前世和自己关系密切，可毕竟今生，连认识都不是认识自己，一旦胡说八道，早晚都会穿帮，与其那样，还不如不说。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  “前辈高人都有些怪脾气，这点我能理解，我今天过来是想找你帮忙，我身受重伤的事，想必颖儿也和你说了，必须使用帝品丹药【回生丹】才能治好，既然你是化云宗无上长老的弟子，不管哪一个长老，都应该和北长老诸葛青云熟悉，如果能委托他帮我炼制这枚丹药，在下感激不尽！”  &nbsp;    &nbsp;    &nbsp;    &nbsp;  玲珑宗主说出了自己的真正来意。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  丹药分为气海级和灵级，其中灵级丹药又分为，下、中、上、绝品、圣品、帝品、天品、道品八个级别。  &nbsp;    &nbsp;    &nbsp;    &nbsp;  整个浮天大陆能炼制帝品丹药的人屈指可数，化云宗也只有东西南北四大无上长老中的北长老丹帝诸葛青云才有如此能力。
